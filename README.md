## 📖 Overview

**JRC-JCoord** is a modern coordinate utility designed for FiveM developers, mappers, and server administrators.

Get precise coordinates, heading information, vector exports, and favorite locations through a sleek and optimized interface.

---

## ✨ Features

### 📍 Live Coordinates

* Real-time X, Y, Z tracking
* Accurate coordinate updates
* Development-friendly display

### 🧭 Heading Detection

* Live heading information
* One-click heading copy
* Perfect for mapping

### 📦 Vector Utilities

* Copy Vector3
* Copy Vector4
* Copy formatted coordinates
* Clipboard support

### ⭐ Favorites System

* Save important locations
* Quick coordinate access
* Easy management

### 🔵 Laser Tool

* Toggle laser visualization
* Precise object placement
* Mapping assistance

### ⚡ Performance

* Lightweight
* Optimized
* Smooth UI
* Low resource usage

## 📦 Export Formats

### Vector3

```lua
vector3(-76.44, -816.47, 326.18)
```

### Vector4

```lua
vector4(-76.44, -816.47, 326.18, 229.89)
```

### Coordinates

```lua
{-76.44, -816.47, 326.18}
```

## 🚀 Installation

1. Download the resource.
2. Place it inside your resources folder.
3. Add the resource to your `server.cfg`.

```cfg
ensure JRC-JCoord
```

4. Restart your server.
